Install PIL on Mac OSX 10.6 (Snow Leopard)

May 10, 2010

  1. Download libjpeg (jpegsrc.v8a.tar.gz) from http://www.ijg.org/files/

    $ ./configure --enable-shared --enable-static
$ make
$ sudo make install

    This will install libjpeg in /usr/local.

  2. Download actual PIL source from http://effbot.org/downloads/#imaging

    $ python setup.py build
$ sudo python setup.py install --optimize=1
